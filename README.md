# Module3_DatabaseIntegration_Handson01_Tejashree



## Overview



This repository contains the solutions for **Hands-On 1 of Module 3 – Database Integration**, completed as part of the **Digital Nurture 5.0 – Python Full Stack Engineer Track**.



The exercises focus on designing, normalizing, and modifying a relational database for a **Student Course Registration System** using **MySQL** and **SQL Data Definition Language (DDL)**.



---



## Tasks Completed



### Task 1



* Created the `college_db` database and all required tables with appropriate constraints.



### Task 2



* Verified database normalization and documented compliance with:



  * First Normal Form (1NF)

  * Second Normal Form (2NF)

  * Third Normal Form (3NF)



### Task 3



* Modified and extended the database schema using `ALTER TABLE` statements.
